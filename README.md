# Snake game on Arduino board

## Current status

** work in progress **

## Installation

	Using Arduino IDE compile and upload code to Arduino.

## Usage

	Use joystick to navigate snack, potentiometer to slow down / speed up.

## Board Curcuit

- Uno R3 controller board
- joystick module
- potentiometer 10k
- LED matrix MAX7219 8x8
- 4 digit display
- push / switch button
